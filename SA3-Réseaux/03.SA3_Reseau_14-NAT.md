<h1 align=center>NAT</h1>

Présentation du Network Address Translation (NAT) et de son rôle dans les réseaux

---
# Introduction

Le **NAT** (Network Address Translation) permet à plusieurs machines d'un réseau local d'accéder à Internet en utilisant une seule adresse IP publique.
- Utilisé dans les box Internet, routeurs de bordure

---
# Routage et Internet

- Les sous-réseaux différents peuvent communiquer via des routeurs
- Internet = interconnexion de réseaux locaux via de nombreux routeurs

---
# Table de routage

- Une **route** est une entrée dans la table de routage d'un routeur
- Permet de définir vers quelle adresse IP un paquet doit être redirigé

---
# Routage statique et par défaut

- Ajout de routes statiques et de routes par défaut pour permettre la communication entre sous-réseaux
- Sur les routeurs Cisco, `0.0.0.0` représente toutes les destinations non couvertes par une route spécifique

---
# Exercice pratique

- Compléter les routes manquantes pour permettre la communication entre tous les sous-réseaux
- Utilisation de la commande `ip route` pour ajouter des routes

---
# Sécurité et NAT

- Le NAT permet de masquer les adresses internes et d'améliorer la sécurité
- Utilisé pour la translation d'adresses lors de l'accès à Internet
