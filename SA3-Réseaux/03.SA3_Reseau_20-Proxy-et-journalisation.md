# Qu'est-ce qu'un proxy ?

Un **proxy** (mandataire) est un composant informatique qui joue le rôle d'intermédiaire entre deux hôtes.
- Généralement un logiciel, intervenant à la couche 7 (application) du modèle OSI
- Non détectable par traceroute (niveau 3)

## Fonctions d'un proxy
- Accélérer la navigation web (cache, compression)
- Journaliser les requêtes (historique, logs)
- Filtrer (blocage de sites ou contenus)
- Rediriger le trafic (proxy inverse)
- Répartir la charge
