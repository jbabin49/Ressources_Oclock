<h1 align=center>Switchs de niveau 3</h1>

Présentation des switchs de niveau 3 et de leur rôle dans le routage inter-VLAN

---
# Switchs de niveau 2

- Les commutateurs (switchs) classiques travaillent à la couche 2 du modèle OSI
- Dirigent les trames Ethernet grâce à l'adresse MAC et la table d'adresses MAC

---
# Switchs de niveau 3

- Certains switchs sont dits "de niveau 3" (Layer 3 switch, L3)
- Travaillent à la couche 3 du modèle OSI, comme les routeurs
- Peuvent effectuer des opérations de routage basiques

## Utilisation typique
- Routage inter-VLAN (faire communiquer plusieurs VLANs sans ajouter de routeur)
- Souvent utilisés pour le cœur de réseau

---
# Configuration de base (Cisco)

- Création de VLANs :
  - `conf t`
  - `vlan 10`
  - `name LAN1`
  - `exit`
  - `vlan 20`
  - `name LAN2`
  - `exit`
  - `vlan 42`
  - `name Management`
  - `exit`
- Configuration d'un lien trunk :
  - `conf t`
  - `interface gigabitEthernet 1/0/1`
  - `switchport trunk encapsulation dot1q`
  - `switchport mode trunk`
  - `switchport trunk native vlan 42`
  - `switchport trunk allowed vlan 10,20,42`
- Création d'interfaces VLAN (passerelles) :
  - `interface vlan 10`
  - `ip address 192.168.1.254 255.255.255.0`
  - `no shutdown`
  - `exit`
  - `interface vlan 20`
  - `ip address 172.16.255.254 255.255.0.0`
  - `no shutdown`
  - `exit`
- Activer le routage :
  - `ip routing`

---
# Pratique

- Les deux LANs peuvent maintenant communiquer via le switch de niveau 3
