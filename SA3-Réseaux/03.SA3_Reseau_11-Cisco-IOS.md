Présentation du système d'exploitation Cisco IOS pour équipements réseau

---
# Qu'est-ce que Cisco IOS ?

**Cisco IOS** (Internetwork Operating System) est le système d'exploitation produit par Cisco Systems pour ses équipements réseau.
- Présent sur la plupart des équipements Cisco

---
# Interfaces d'administration

Cisco IOS propose toujours au moins une interface homme-machine :
- Interface en ligne de commande (**CLI**)
- Parfois une interface web graphique

## Accès à la CLI
- Liaison série (port Console)
- À distance via **Telnet**
- À distance via **SSH**

Par défaut, seule la liaison série est activée.

---
# Liaison série et câbles

- Utilisation du protocole RS-232 (EIA/TIA-232)
- Adaptateur DB-9 vers USB souvent nécessaire
- Les équipements récents utilisent un port USB console

---
# Connexion et configuration

- Utilisation d'un logiciel comme **PuTTY** pour la connexion série
- Identification du port COM sous Windows
- Paramétrage : vitesse 9600 bauds

---
# Utilité de la connexion

- Permet d'activer/configurer des fonctionnalités avancées sur les switches et routeurs

---
# Fichiers de configuration

- **running-config** : configuration active (en RAM, perdue à l'extinction)
- **startup-config** : configuration de démarrage (en mémoire non-volatile)

Au démarrage, le fichier startup-config est copié en running-config.
